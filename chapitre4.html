L’introduction :

Le www est un system hypertext public sur internet , qui permet au utilisateurs d’accéder et de visualiser des pages et des lien par le biais d’hyperlienle web a été inventé après internet . Uniform resource locator (URL) est un nom unique pour une page qui décrit l’emplacement d’une ressource et il contient l’adresse du serveur et la page sur le serveur . Le texte explique également les trois mécanismes de base du web ( URL , http , paradigme cient-serveur)

Une page web est un document multimédia qui peut contenir du texte, des images et des éléments interactifs , le site web aussi est un ensemble des pages web reliées elles par des hyperliens et structurés sous une forme arborescente HTML, HTM ou bien XHTML, HTML5, sont des langages pour décrire une page web et ces fichiers sont stockés au format .html, .xhtml …

Le document html est un fichier texte construit avec une syntaxe spécifique , il est structuré avec des balises et affiché dans un navigateur web (Mozilla firefox, safari..) Les principes de HTML

html: marque le début et la fin du document

title : spécifie le titre du document

body : le contenu principal (paragraphes..)
